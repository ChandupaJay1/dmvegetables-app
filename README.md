## âš™ï¸ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone git@github.com:ChandupaJay1/dmvegetables-app.git
   cd dmvegetables-app
   ```

2. **Install Composer dependencies:**
   ```bash
   composer install
   ```

3. **Install NPM dependencies:**
   ```bash
   npm install
   ```

4. **Environment Setup:**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. **Run the development server:**
   ```bash
   npm run dev
   # In a separate terminal
   php artisan serve
   ```

6. **Build for production:**
   ```bash
   npm run build
   ```
